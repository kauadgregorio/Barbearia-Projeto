# Barbearia-Projeto

Site institucional e sistema de agendamento desenvolvido para a Barbearia Vitinho, localizada em Taubaté - SP.
O projeto nasceu como uma solução real para um pequeno negócio, unindo presença digital profissional com praticidade no agendamento de serviços.

*Site Institucional:*
 Página inicial com identidade visual da marca
 Seção de serviços e preços
 Galeria de cortes
 Localização e contato
 Botão de WhatsApp flutuante
 Design responsivo (mobile e desktop)

*Sistema de Agendamento:*
 Formulário de agendamento integrado ao site
 Escolha de serviço, data e horário
 Confirmação via WhatsApp
 Painel administrativo para o barbeiro
 Visualização de agendamentos do dia

 *Tecnologias Utilizadas:*
Frontend: HTML5, CSS3, JavaScript
Backend: Python
Banco de Dados: SQLite3
